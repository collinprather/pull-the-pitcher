query_statcast[source]

query_statcast(start_dt:"Beginning date to pull data from"=None, end_dt:"End date to pull data from"=None, team:"Abbreviation for team of interest"=None, verbose:"Whether or not to print verbose updates"=True, output_type:"What format to save data in"='db', overwrite:"Whether or not to overwrite the db table if it already exists"=True, output_path:"path to location that data should be saved"='.')

Note, working with sqlite files, as opposed to csv files is the recommended usage.

{% endraw %}

Usage

From the command-line 

$ query_statcast --start_dt 2019-05-07 --end_dt 2019-06-09 --output_type db --output_path /tmp
