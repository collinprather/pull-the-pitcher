query_statcast[source]

query_statcast(start_dt:"Beginning date to pull data from"=None, end_dt:"End date to pull data from"=None, team:"Abbreviation for team of interest"=None, verbose:"Whether or not to print verbose updates"=True, if_exists:"How to behave if the table already exists."='fail')

Callable from the command-line or in Python. Pulls pitch-level MLB data from statcast. Saves to a pre-instantiated Postgres DB in AWS RDS.

  • inputs:

    • start_dt: str, Beginning date to pull data from = None
    • end_dt: str, End date to pull data from = None
    • team: str, abbreviation for team of interest = None
    • verbose: bool, Whether or not to print verbose updates
    • if_exists: str, How to behave if the table already exists.
      • 'fail': Raise a ValueError.
      • 'replace': Drop the table before inserting new values.
      • 'append': Insert new values to the existing table.

  • outputs:

    • None
{% endraw %} {% raw %} 
# query_statcast tests


# getting red sox data from July 7th, 2019
start_dt = end_dt = "2019-07-07"
query_statcast(
    start_dt=start_dt,
    end_dt=end_dt,
    team="BOS",
    if_exists="replace",
)

# ensuring David Price threw 99 pitches
engine = load_postgres_engine()
with engine.connect() as connection:
    result = connection.execute("""select count(1)
                                   from statcast_2019
                                   where player_name = 'David Price'""")
assert result.first()[0] == 99

query_db[source]

query_db(year:str='2019', columns:str=`''*, **limit**:int=*None*, **verbose**:bool=*True`*)

Queries a sqlite db file. Assumes that it's been created by query_statcast. Only queries for a single year at a time.

  • intputs:

    • year: str, year of data to query
    • columns: str, which columns from the statcast data to include in table
    • limit: int, the maximum number of rows to retrieve (postgresql documentation)
    • verbose: bool, Whether or not to print verbose updates

  • output:

    • df: pd.DataFrame, DataFrame populated with data queried from database
