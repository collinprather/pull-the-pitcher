query_statcast[source]

query_statcast(start_dt:"Beginning date to pull data from"=None, end_dt:"End date to pull data from"=None, team:"Abbreviation for team of interest"=None, verbose:"Whether or not to print verbose updates"=True, output_type:"What format to save data in"='db', overwrite:"Whether or not to overwrite the db table if it already exists"=False, output_path:"path to location that data should be saved"='.')

Callable from the command-line or in Python. Pulls pitch-level MLB data from statcast. Saves as either a sqlite db file, or csv.

  • inputs:

    • start_dt: str, Beginning date to pull data from = None
    • end_dt: str, End date to pull data from = None
    • team: str, abbreviation for team of interest = None
    • verbose: bool, Whether or not to print verbose updates
    • output_type: str, What format to save data in (must be one of {'db', 'csv'}) = 'db'
    • overwrite: bool, Whether or not to overwrite the db table if it already exists = False
    • output_path: str, Path to the location that the data should be saved at = '.'

  • outputs:

    • None

# output type must be db or csv

query_db[source]

query_db(db_path:str='../data/raw/statcast_pitches.db', year:str='2019', columns:str=`''*, **limit**:int=*None*, **verbose**:bool=*True`*)

Queries a sqlite db file. Assumes that it's been created by query_statcast. Only queries for a single year at a time.

  • intputs:

    • db_path: str, path that db file is located at
    • year: str, year of data to query
    • columns: str, which columns from the statcast data to include in table
    • limit: int, the maximum number of rows to retrieve (postgresql documentation)
    • verbose: bool, Whether or not to print verbose updates

  • output:

    • df: pd.DataFrame, DataFrame populated with data queried from database
